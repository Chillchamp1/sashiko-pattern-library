Pattern Library

A free, open-source space for the crafting and visible mending community.

This library is built with love to keep traditional patterns alive and to support sustainable crafting and mindful garment repair.

Feel free to copy, mod or reimagine this project. If you do so, please mention sashikolib.org as a source.

If you love what’s being built here, a small donation keeps the servers running and the stitches flowing❤️
